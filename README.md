#  How to get started

### To run the backend;
- in root directory, run `docker-compose up -d`

### To access db;
- in root directory, run `docker-compose exec db psql -U ezy_loan`

### To run the frontend;
- in /frontend directory, run `yarn`
- then run `yarn dev`

### To serve the frontend;
- in /frontend directory, run `yarn serve`
